# LGUB 무엇이든 만들어주기 (SDK 기반 즉석 개발)

하드웨어팀이 기존 스킬로 안 되는 것을 말하면, **직접 만들어서 돌려준다.** 사용자는
"무엇을 원하는지"만 말하면 되고, 구현·빌드·실행·검증은 전부 이 쪽에서 한다.

## 절대 원칙
1. **반드시 SDK(`lorddom::LgubSensor`) 위에서 만든다.** 시리얼/Modbus/CRC 를 다시 짜지
   말고 `read_distance()`, `read_registers()`, `autodetect_port()`, 자동재접속·무효필터를
   그대로 재사용한다.
2. **기존 자산을 먼저 재사용/변형한다.** 새로 짜기 전에 이미 있는 걸 확인:
   - CLI 서브커맨드: `tools/lgub_cli.cpp` (measure/monitor/stream/log/doctor)
   - 웹 대시보드: `tools/lgub_webapp.cpp`
   - 예제: `examples/read_distance.cpp`
   요청이 이들의 작은 변형이면 복제·수정이 가장 빠르다.
3. **읽기 전용이 기본.** 슬레이브 주소·보드레이트 변경 등 쓰기(write)는 잘못하면 통신이
   끊기므로, 사용자가 명시적으로 요청하고 위험을 확인한 뒤에만.

## 작업 흐름
1. **요청 구체화**: 애매하면 1~2개만 짧게 되묻는다(예: 임계값 몇 cm? 소리인지 화면인지?
   기록 파일 필요한지?). 비전문가이므로 전문용어는 피한다.
2. **형태 결정**:
   - 단순 로직/조건(예: "20cm보다 가까우면 경고") → 작은 CLI 프로그램(`tools/` 또는 `examples/`)
   - 화면이 필요(대시보드/버튼/알림 색) → `lgub_webapp.cpp` 를 복제·확장한 웹앱
   - 값 소비/연동(다른 프로그램에 전달) → JSON/CSV 출력 도구
3. **구현**: 새 `.cpp` 를 추가하고 `CMakeLists.txt` 에 타깃을 등록한다(기존 타깃 패턴 복사).
4. **빌드**: `cmake -S . -B build >/dev/null && cmake --build build -j`
5. **검증(필수)**: 실제로 실행해 의도대로 동작하는지 관찰한다(값이 나오는지, 조건이
   맞을 때 반응하는지). 하드웨어가 없으면 무-센서 경로라도 확인하고 그 사실을 밝힌다.
6. **보고**: 무엇을 만들었고 어떻게 쓰는지 평이한 한국어로 요약. 실행 방법이 터미널을
   요구하면, 비전문가용으로는 Claude가 대신 실행해 주거나 웹앱 형태를 제안한다.
7. **저장(커밋 금지 기본)**: 요청으로 만든 프로그램/앱은 **커밋·푸시하지 않는다.**
   파일은 작업트리에 두고 빌드·실행만 한다. 사용자가 "이건 커밋해줘"라고 명시할 때만 커밋한다.
   (SDK 코어·기존 스킬 수정 등 인프라 변경은 별개 — 그건 요청에 따른다.)

## 실행 요청 대응 ("실행해줘/돌려줘/켜줘")
만든 뒤 실행을 요청하면 **Claude가 직접 실행**한다(사용자 터미널 조작 불필요):
- 짧게 끝나는 것 → 바로 실행하고 결과를 평이하게 보고.
- 계속 도는 것(감시 루프·서버) → **백그라운드로 실행**하고 상태를 알린다. "그만/꺼줘" 하면 종료.
- 웹앱류 → 백그라운드 실행 + `xdg-open` 으로 브라우저까지 열어준다.
- 이미 만들어 둔 것을 다시 "실행해줘" 하면, 재빌드 없이 (필요시 빌드 후) 바로 실행한다.
- 종료/상태 확인은 정확 매칭 사용(`ps -C <name>`, 포트 `ss -ltn`). `pgrep -f` 는 검사
  명령 자신을 잡는 오탐 주의.

## 품질 수칙
- 새 프로그램도 자동탐지·자동재접속·무효(NoTarget/OutOfRange) 처리를 누리도록 SDK 흐름을
  탄다(값이 0이면 "대상 없음"이지 0m 가 아니다).
- 실시간 스트림 출력은 `fflush(stdout)` 로 즉시 반영.
- 결과 수치는 m·cm 를 함께 보여준다.
- 오래 도는 것은 백그라운드로 실행하고 상태를 요약한다.

## 예시 매핑
| 요청 | 만들 것 |
|---|---|
| "20cm보다 가까워지면 경고해줘" | 임계값 감시 CLI(조건 충족 시 메시지/종료코드), 필요시 웹앱에 빨간불 |
| "5번 재서 평균 내줘" | N회 측정 평균 CLI |
| "가까워지면 화면이 빨개지는 앱" | lgub_webapp 복제 + 임계값 색상 로직 |
| "조건 될 때만 CSV로 남겨줘" | log 변형(임계 이벤트만 기록) |
